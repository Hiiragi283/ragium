# Materials

## Raginite

ラギナイトとは，地下に生成される鉱物の一種である。

外観はマゼンタ色の粉末状であり，レッドストーンと混同されることがあるが，その物性はレッドストーンと大きく異なる。

金属にラギナイト粉末を添加することにより，熱伝導性および電気伝導性を大きく向上させる。 一般に安価な銅との合金が使われるが，金との合金も用いられている。

また，半導体の不純物としてダイヤモンドに添加されることがある。

## Azure Shard

紺碧の欠片とは，人工的に作られるアメジストとラピスラズリの固溶体である。

外観は淡青色の結晶であり，「青い水晶」という名で呼ばれることもある。

固有の結晶構造をもち，これを鉄に添加した「紺鉄」は鉄単体と比べて強度が向上している。

## Deep Steel

深層鋼とは，紺鉄と共振の残骸の暗青色の合金である。

共振の残骸は，固有の気候をもつ地下深くに生成されるほか，特定の生物が死亡する時に発する周波数に共鳴して露出することがある。

この合金は紺鉄より遥かに硬度が高く，耐衝撃性にも優れている。これは，受けた衝撃に対して自身の構造を逆位相で振動させる特性によるものだと考えられている。

## Crimson Crystal

深紅の結晶とは，深紅の樹木から精製される人工の結晶である。

外観は深紅色の六角柱状である。

自然に生成されることはほとんどなく，人工的な蒸留による精製の成果として得られる。

この物質は燃料として優秀であり，直接燃やすだけでなく液体燃料に溶かすことでその燃焼効率を向上させることできる。

## Warped Crystal

歪んだ結晶とは，歪んだ樹木から精製される人工の結晶である。

外観は青緑色の六角柱状である。

自然に生成されることはほとんどなく，人工的な蒸留による精製の成果として得られる。

この物質の結晶構造は三次元のみで説明ができず，四次元あるいはそれ以上の次元に渡って結晶を形成しているといわれている。

このことから，空間転移などへの応用が期待されている。

## Eldritch Pearl

異質な真珠とは，深紅の結晶と歪んだ結晶から合成される紫色の球体である。

この物質の詳しい性質は解明されておらず，現在も分析が進められている。

深紅の結晶に由来するエネルギーによって，歪んだ結晶を介して別次元から未知のエネルギーを取り出していることが確認されており，この特性は溶融状態でも維持される。

このエネルギーは接触した物体を変異させることがわかっており，変異後は元の物体に近い場合もあれば，全く異なるものになる場合もあるため，法則性は解明されていない。
